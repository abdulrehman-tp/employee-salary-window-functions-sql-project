# SQL Window Functions Analysis

## Project Overview
This project focuses on analyzing employee salary data using SQL Window Functions.  
It includes ranking functions, department-wise analysis, running totals, and advanced analytical queries commonly used in Data Analyst roles.

The goal of this project is to strengthen SQL analytical skills using real-world business-style questions.

---

## Dataset Description

Table Name: emp_salary

Columns:
- emp_id
- emp_name
- department
- salary

---

## Topics Covered

- Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- PARTITION BY
- OVER()
- Running Totals
- Department-wise Ranking
- Top N Analysis
- CTE (Common Table Expressions)

---

## Business Questions Solved

### Basic Level
1. Show salary rank of all employees  
2. Show highest salary employee in each department  
3. Show second highest salary in each department  

---

### Medium Level
4. Show department average salary beside every employee  
5. Show running total of salaries  
6. Show employees ranked by salary inside department  

---

### Advanced Level
7. Difference between RANK() and DENSE_RANK()  
8. Show top 2 highest paid employees from each department  
9. Show employees earning above department average using window function
